Hilfe

Anlegen von Tracks

Track werden durch einfaches Klicken in die Karte angelegt. Die aktuelle Länger erscheint über dem Cursor. Durch einen Doppelklick wird der Track abgeschlossen. Er erscheint dann in einer durchgezogenen Linie.

Löschen der letzten Punkte

Durch drücken der Backspace-Taste wird der jeweils letzte Punkt aus dem Track gelöscht.

Bearbeiten von Tracks

Um den Track zu bearbeiten muss er erst durch einen Doppelklick abgeschlossen werden. Danach kann er mit der Maus angeklickt werden. Es wird dann heller dargestellt. Wenn sie mit der Maus in die Nähe einer Linie kommen erscheint ein Punkt, ein Drag and Drop können sie so eine Linie halbieren und einen neuen Punkt erzeugen. Wenn an eine Ecke, bzw an einen bestehenden Punkt auf der Linie kommen wird der Punkt gesnappt und die können den bereit vorhandenen Punkt verschieben. Punkte zu löschen ist im Bearbeitenmodus aktuell noch nicht möglich.
